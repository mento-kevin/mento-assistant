# 🌐 自定义域名部署指南（Netlify + 阿里云/腾讯云）

## 📋 整体流程

1. 在Netlify部署网站并获取临时网址
2. 在Netlify设置自定义域名
3. 在阿里云/腾讯云配置DNS解析
4. 等待生效（约5-30分钟）
5. 网站上线！

---

## 第一步：在Netlify部署并获取网站地址

### 1.1 部署网站

按照之前的步骤，把dist文件夹部署到Netlify：
- 访问：https://app.netlify.com
- 注册/登录
- 拖拽dist文件夹部署

### 1.2 记录Netlify分配的网址

部署成功后，你会看到一个临时网址，例如：
```
https://opc-assistant-xxxxx.netlify.app
```

把这个网址记录下来，后面配置DNS时会用到。

---

## 第二步：在Netlify添加自定义域名

### 2.1 进入域名设置

1. 在Netlify控制台，点击你的网站
2. 点击 **"Site settings"**
3. 找到 **"Custom domains"** 部分
4. 点击 **"Add custom domain"**

### 2.2 输入你的域名

输入你要绑定的域名，例如：
```
www.yourdomain.com
```
（或者只用主域名：`yourdomain.com`）

### 2.3 选择域名选项

Netlify会问你：
- **Add both www and non-www**
- **Only add www**
- **Only add non-www**

**推荐选择**：Add both（同时添加带www和不带www的）

### 2.4 记录Netlify提供的DNS验证信息

Netlify会显示你需要配置的DNS记录，类似：
```
Type    Name    Value
A       @       75.2.12.34
CNAME   www     your-site-name.netlify.app
```

把这些记录记下来，接下来要在阿里云/腾讯云配置。

---

## 第三步：在阿里云配置DNS（推荐）

### 3.1 登录阿里云域名控制台

1. 访问：https://dns.console.aliyun.com
2. 登录你的阿里云账号
3. 进入 **"云解析DNS"**

### 3.2 选择你的域名

在域名列表中找到你要绑定的域名，点击 **"解析设置"**

### 3.3 添加DNS记录

#### 如果要绑定主域名（yourdomain.com）

点击 **"添加记录"**，添加以下记录：

**记录1 - A记录（必须）**
```
记录类型：A记录
主机记录：@
记录值：75.2.12.34（Netlify提供的IP）
TTL：10分钟
```

**记录2 - CNAME记录（可选，用于www）**
```
记录类型：CNAME
主机记录：www
记录值：your-site-name.netlify.app（你的Netlify临时网址）
TTL：10分钟
```

#### 如果只绑定www子域名

只需要添加一条CNAME记录：
```
记录类型：CNAME
主机记录：www
记录值：your-site-name.netlify.app
TTL：10分钟
```

### 3.4 添加Netlify验证记录（可选但推荐）

Netlify可能要求添加验证记录来确认域名所有权：

```
记录类型：TXT
主机记录：_netlify
记录值：netlify-site-verification=xxxxxx（Netlify提供的值）
TTL：10分钟
```

---

## 第三步：在腾讯云配置DNS（备选）

### 3.1 登录腾讯云域名控制台

1. 访问：https://console.cloud.tencent.com/domain
2. 登录你的腾讯云账号
3. 进入 **"我的域名"**

### 3.2 选择你的域名，点击 "解析"**

### 3.3 添加DNS记录

点击 **"添加记录"**，配置同上阿里云的步骤。

---

## 第四步：等待DNS生效

### 4.1 生效时间

- **阿里云/腾讯云**：通常5-10分钟生效
- **全球传播**：最多需要24-48小时（但一般1小时内）

### 4.2 验证DNS配置

访问：https://tool.chinaz.com/dns

输入你的域名，检查DNS是否解析到正确的IP地址。

---

## 第五步：在Netlify验证并启用HTTPS

### 5.1 检查域名验证状态

回到Netlify的Custom domains设置页面，你会看到域名状态：
- ✅ "Netlify DNS" 或 "Verification OK"
- ⏳ "Waiting for DNS propagation"

### 5.2 启用免费的SSL证书（重要！）

1. 在Custom domains页面，点击 **"HTTPS"** 部分
2. 点击 **"Verify DNS configuration"**
3. Netlify会自动申请Let's Encrypt免费SSL证书
4. 等待证书签发（通常5-10分钟）

### 5.3 强制HTTPS

证书生效后，勾选：
☑️ **Enforce HTTPS**

这样所有HTTP请求都会自动跳转到HTTPS。

## 📝 常见问题

### Q1: DNS配置后网站打不开？
- 等待5-30分钟让DNS完全生效
- 清除浏览器缓存
- 使用 https:// 访问（而不是 http://）

### Q2: SSL证书申请失败？
- 确认DNS已经正确配置并生效
- 确认域名已经备案（国内域名必须备案才能用SSL）
- 等待10分钟后重试

### Q3: 只想用主域名，不用www？
1. 在Netlify选择 "Only add non-www"
2. DNS只配置A记录，不配置CNAME

### Q4: 阿里云/腾讯云需要备案吗？
- **国内服务器**：必须备案
- **Netlify**：海外服务器，不需要备案
- 如果域名已备案，用Netlify没问题；如果没备案，建议用海外域名商
