# OPC Assistant - 创业助手

一个帮助创业者规划、管理和复盘的一站式平台。

## ✨ 特性

- 🗺️ **创业规划问卷** - AI 驱动的 90 天创业规划
- 📊 **数据仪表盘** - 实时掌控销售和财务数据
- 📝 **每日复盘** - AI 辅助的成长记录与分析
- 💰 **财务中心** - 客户、合同、发票管理
- 💡 **知识库** - 灵感收集与整理
- 🤖 **AI 助手** - 多模型支持（GPT、Claude、通义千问等）

## 🎨 设计

采用简约高级的黑白灰 + 嫩绿色配色方案，呈现专业、现代的视觉效果。

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript
- **UI 组件库**: Material-UI (MUI)
- **状态管理**: Zustand
- **路由**: React Router
- **构建工具**: Vite
- **图表**: ECharts

## 📦 安装

```bash
# 克隆仓库
git clone <your-repo-url>
cd opc-assistant

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 🚀 部署

本项目已配置 Vercel 一键部署：

1. Fork 本仓库
2. 在 [Vercel](https://vercel.com) 导入项目
3. 自动部署！

或手动构建：

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

构建产物在 `dist` 目录，可部署到任意静态服务器。
