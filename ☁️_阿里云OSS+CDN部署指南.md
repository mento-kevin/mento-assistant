# ☁️ 阿里云 OSS + CDN 部署详细指南（国内访问超快！）

## 📋 整体流程

1. 注册/登录阿里云
2. 创建 OSS Bucket 并配置静态网站
3. 上传 dist 文件夹
4. 购买 CDN 加速
5. 配置 CDN 回源到 OSS
6. 绑定自定义域名并配置 DNS
7. 申请免费 SSL 证书
8. 网站上线！

---

## 第一步：登录阿里云控制台

### 1.1 访问阿里云
打开浏览器，访问：https://www.aliyun.com

### 1.2 登录
- 如果有账号，点击右上角"登录"
- 如果没有账号，点击"免费注册"（支持支付宝/手机号/邮箱）

---

## 第二步：创建 OSS Bucket

### 2.1 进入 OSS 控制台
1. 登录后，在搜索框输入"OSS"
2. 点击 **"对象存储 OSS"**
3. 进入 OSS 控制台

### 2.2 创建 Bucket
1. 点击左侧 **"Bucket 列表"**
2. 点击右上角 **"创建 Bucket"**
3. 填写配置：

```
Bucket 名称：opc-assistant（或你喜欢的名字）
地域：建议选 华东1（杭州） 或 离你近的
存储类型：标准存储
读写权限：公共读
版本控制：不开通
日志管理：关闭
```

4. 点击 **"确定"** 创建

---

## 第三步：上传网站文件

### 3.1 进入 Bucket 管理
1. 点击刚才创建的 Bucket 名称（比如 opc-assistant）
2. 进入 **"文件管理"**

### 3.2 上传 dist 文件夹
1. 点击 **"上传文件"**
2. 打开文件夹：`c:\Users\79849\Desktop\opc\opc-assistant\dist`
3. 选择所有文件和文件夹：
   - `index.html`
   - `assets/` 文件夹
4. 拖拽到上传区域，或点击"选择文件"
5. 点击 **"上传文件"**

**注意**：确保把 dist 文件夹里的内容直接上传，不要上传 dist 文件夹本身！

---

## 第四步：配置 OSS 静态网站托管

### 4.1 进入静态页面设置
1. 在 Bucket 管理左侧菜单，点击 **"数据管理"** → **"静态页面"**
2. 点击 **"设置"**

### 4.2 配置首页和 404
填写：
```
默认首页（index document）：index.html
默认 404 页（error document）：index.html
```
（因为是 SPA 单页应用，所有路径都返回 index.html）

3. 点击 **"确定"**

### 4.3 测试临时访问地址
保存后，你会看到一个访问地址，类似：
```
http://opc-assistant.oss-cn-hangzhou.aliyuncs.com
```

在浏览器访问这个地址，确认网站正常显示！

---

## 第五步：购买 CDN 加速

### 5.1 进入 CDN 控制台
1. 回到阿里云首页，搜索"CDN"
2. 点击 **"CDN 内容分发网络"**
3. 进入 CDN 控制台

### 5.2 添加 CDN 域名
1. 点击左侧 **"域名管理"**
2. 点击 **"添加域名"**

### 5.3 配置 CDN 源站
填写：
```
加速域名：www.mento.net.cn（你的域名）
业务类型：图片小文件
源站类型：OSS域名
源站域名：选择你的 OSS Bucket 域名（类似 opc-assistant.oss-cn-hangzhou.aliyuncs.com）
```
4. 点击 **"下一步"**

### 5.4 选择计费方式
选择 **"按流量计费"**（免费版一般够用）
点击 **"确认添加"**

---

## 第六步：配置 DNS

### 6.1 获取 CDN 的 CNAME 地址
添加成功后，在 CDN 域名管理列表中：
找到你的域名，右侧会显示 CNAME 地址，类似：
```
www.mento.net.cn.w.kunlunle.com
```

### 6.2 在阿里云 DNS 解析里配置
1. 进入阿里云 **"云解析 DNS"** 控制台
2. 找到你的域名（mento.net.cn）
3. 点击 **"解析设置"**

### 6.3 添加 CNAME 记录
点击 **"添加记录"**，填写：
```
记录类型：CNAME
主机记录：www（或 @ 如果用主域名）
记录值：刚才 CDN 给你的 CNAME 地址
TTL：10分钟
```

---

## 第七步：申请免费 SSL 证书（必须！）

### 7.1 进入 SSL 证书控制台
1. 阿里云搜索"SSL证书"
2. 进入 **"SSL 证书（应用安全）"**

### 7.2 申请免费证书
1. 点击 **"证书购买"**
2. 选择 **"免费证书"** → **"DigiCert 免费版 SSL"**
3. 点击 **"立即购买"**
4. 确认订单，点击 **"购买"**（0元）

### 7.3 提交证书申请
1. 回到 **"我的证书"**
2. 点击刚才购买的免费证书后面的 **"申请"**
3. 填写信息：
```
证书绑定域名：www.mento.net.cn（你的域名）
个人信息按真实填写
域名验证方式：DNS验证
```
4. 点击 **"下一步"**

### 7.4 DNS 验证（自动或手动）
阿里云会自动帮你添加 TXT 验证记录，或者：
- 如果需要手动，按提示添加 TXT 记录

### 7.5 下载并部署证书
1. 证书签发后（一般几分钟）
2. 点击 **"下载证书"**
3. 选择 **"Nginx"** 格式下载
4. 下载后得到压缩包，解压后有两个文件：
   - `xxxx.pem`（证书文件）
   - `xxxx.key`（私钥文件）

### 7.6 在 CDN 里部署证书
1. 回到 CDN 控制台
2. 点击你的域名
3. 点击 **"HTTPS 设置"** → **"证书配置"**
4. 点击 **"上传自定义证书"**
5. 把刚才下载的证书内容和私钥内容粘贴进去
6. 点击 **"确定"**

---

## 第八步：配置强制 HTTPS

### 8.1 在 CDN 开启
1. 在 CDN 域名设置里
2. 找到 **"HTTPS 设置"** → **"强制跳转 HTTPS"**
3. 开启：**"HTTP 301 跳转 HTTPS"**

---

## 第九步：等待生效

### 9.1 DNS 生效时间
- 阿里云 DNS：通常 5-10 分钟生效
- 全球传播：最多 24 小时

### 9.2 测试访问
访问你的域名：
```
https://www.mento.net.cn
```

---

## 🎉 完成！

恭喜！你的网站已经部署到阿里云 OSS + CDN 了！
