## 📝 上传到 GitHub 的步骤

### 步骤1：访问你的仓库

创建仓库后会进入仓库页面，URL类似：
```
https://github.com/你的用户名/opc-assistant
```

### 步骤2：上传文件

1. 点击 "uploading an existing file" 链接
2. 在文件夹中导航到：`c:\Users\79849\Desktop\opc\opc-assistant`
3. 按住 Ctrl 键，点击选择以下文件/文件夹：
   - src
   - public（如果有）
   - package.json
   - package-lock.json
   - vite.config.ts
   - tsconfig.json
   - tsconfig.node.json
   - tailwind.config.ts
   - postcss.config.js
   - index.html
   - .gitignore
   - vercel.json
   - README.md
   - DEPLOYMENT.md
   - DEPLOY_STEP_BY_STEP.md
   - RELEASE_CHECKLIST.md
   - .env.example
4. 拖拽到 GitHub 页面
5. 点击 "Commit changes"

### 步骤3：验证上传

上传成功后，刷新仓库页面，应该能看到：
- ✅ src 文件夹

## 🚀 下一步：部署到 Vercel

完成上传后，前往 Vercel：

1. 访问 https://vercel.com
2. 登录（推荐用 GitHub 账号）
3. 点击 "Add New..." → "Project"
4. 导入你的 opc-assistant 仓库
5. 点击 "Deploy"
6. 等待部署完成（1-2分钟）
7. 获得你的网站 URL！🎉
